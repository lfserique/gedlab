# Documentacao Complementar

Esta pasta foi criada para armazenar materiais adicionais relacionados ao projeto GEDLab.

## Sugestao de Organizacao

- `screenshots/`: capturas de tela da aplicacao
- `diagramas/`: diagramas de arquitetura, casos de uso ou fluxo
- `relatorios/`: textos tecnicos, entregas academicas ou memoriais
- `apresentacao/`: material de apoio para demonstracoes

## Sugestao de Conteudo

- imagem da tela de login
- imagem do painel principal
- imagem do upload de documento
- imagem da auditoria em blockchain
- imagem do processo de assinatura digital

Caso deseje publicar imagens no README principal, recomenda-se criar a subpasta `screenshots/` e referenciar os arquivos a partir dela.
